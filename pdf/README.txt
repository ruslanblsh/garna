Garna for Creators - PDF version

Open index.html in a browser and use Print -> Save as PDF.

What is different from the main landing page:
- No CTA buttons.
- No language switcher in the header.
- No FAQ section.
- No separate How it works section.
- What Garna helps with uses compact square cards.
- Platform ticker is static.
- Contacts are emphasized near the end and in the footer.

Manual editing:
- Text translations are in script.js, inside the i18n dictionary.
- PDF-specific visual overrides are at the bottom of styles.css under "PDF VERSION OVERRIDES".
